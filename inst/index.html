RKML

The RFirefox framework currently includes both an R package and a Firefox extension. With these two pieces of software, it allows the user to create web pages which contain and can excecute R code. Furthermore, it allows both the R engine and the Javascript engine to be aware of and communicate with eachother.

Documentation

Documentation for RFirefox is largely non-existent. This will change over the coming weeks, but for now see the tests and example webpages which ship with the package (RFirefox/inst/tests/*.html and RFirefox/inst/examples/*.hml, respectively). These should all work and give you a good idea of how to do some of the basic things RFirefox is designed for.
